## 📝 注意事项

1. **必须通过HTTP服务器运行**
   - 直接打开HTML文件会报错
   - 必须使用 `start.bat` 或 `start.sh`

2. **数据文件**
   - 当前 `noun.json` 只包含6个示例单词
   - 完整数据可以从 `riYu_SIJI/data/words.json` 复制

3. **浏览器兼容性**
   - 需要现代浏览器（Chrome, Firefox, Safari, Edge）
   - 需要支持ES6和Fetch API
